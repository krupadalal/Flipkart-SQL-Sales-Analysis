🛍️ Flipkart Sales Analysis using SQL

📊 Project Overview

This project analyzes Flipkart sales data using SQL to generate insights related to customers, products, and orders.

🛠 Tools Used

* SQL (MySQL)
* MySQL Workbench

📂 Dataset Details

* 100,000 rows
* 14 columns
* Includes Customers, Products, and Orders data

📈 Key Analysis Performed

👤 Customer Insights

* Customer segmentation (Age-wise)
* Top customers by revenue
* City-wise high-value customers
* Orders by email domain with revenue

📦 Product Insights

* Sales by product category
* Top-selling products
* Most sold category
* Top cancelled products
* Products in transit

📦 Order Insights

* Order status distribution
* Top 10 highest order days
* Year-wise total orders
* Year-wise revenue

⚡ Advanced SQL (Window Functions)

* RANK() for product revenue ranking
* DENSE_RANK() for customer ranking
* ROW_NUMBER() for product sales

📂 Files Included

* SQL Scripts
* Dataset
* Project Presentation (PPT)

🚀 How to Use

1. Import dataset into MySQL
2. Run SQL scripts step-by-step
3. Analyze results

📌 Conclusion

This project demonstrates how SQL can be used to extract meaningful business insights and support data-driven decision-making.
